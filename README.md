# submodule-yjs
助手--研究生端服务--小程序版(不涉及数据库, 加密验证等操作)


## 这本项目只提供小程序接口, 相当于仅仅是个爬虫, 封装json数据 
  重点看一下GraduateDept爬虫类, 了解一下爬虫的基本方法和流程, 利用httpclient和xpath获取解析数据😔, 利用opencv和tess4j解析验证码(现在正确率85~90%左右, 我更改了一下实现图片处理方法,效果还可以)
